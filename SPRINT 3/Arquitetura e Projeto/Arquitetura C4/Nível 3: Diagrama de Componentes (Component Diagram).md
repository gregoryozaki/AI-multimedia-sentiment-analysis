## Nível 3: Diagrama de Componentes (Component Diagram) ##

O Diagrama de Componentes aprofunda a visão de um único contêiner (selecionado do nível 2), mostrando os componentes que residem dentro dele. Um componente é um agrupamento de código bem definido que entrega um conjunto de funcionalidades específicas, geralmente com uma interface clara. Este diagrama é mais relevante para desenvolvedores, respondendo: "Quais são os principais blocos de construção dentro deste contêiner e como eles colaboram?"
